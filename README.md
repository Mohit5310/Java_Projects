## 📂 Featured Projects

### 1. Vroomz (Car Rental Application) 🚗
A scalable, modern car rental platform built using a **Microservices Architecture**. 
* **Key Features:**
  * Decentralized services registered on a **Eureka Discovery Server**.
  * Secure user authentication and authorization using **JWT (JSON Web Tokens)**.
  * Independent services for user management, car inventory, and booking handling.
* **Tech Stack:** Spring Boot, Spring Cloud, Spring Security, JWT, Hibernate/JPA, MySQL.

### 2. Attendance Management System 📋
A robust desktop application designed to automate and streamline attendance tracking for organizations or educational institutions.
* **Key Features:**
  * **Interactive Desktop GUI:** User-friendly interface built entirely using Java Swing components.
  * **Role-Based Login:** Secure access control for Admin/Faculty to manage records and Students/Employees to view data.
  * **Database Integration:** Real-time data persistence, retrieval, and automated attendance logging using MySQL.
  * **Search & Filter:** Advanced filtering options to generate attendance reports based on dates, classes, or individual IDs.
* **Tech Stack:** Java (Core & Advanced), Java Swing (GUI), JDBC, MySQL Database.
